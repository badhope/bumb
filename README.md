# ☢️ Nuclear Weapon Simulator

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Version](https://img.shields.io/badge/version-3.0.0-green.svg)]()
[![Platform](https://img.shields.io/badge/platform-Web-orange.svg)]()
[![Deploy](https://img.shields.io/badge/deploy-GitHub%20Pages-blue.svg)]()

一个基于物理模型的核武器效应模拟器，用于教育和研究目的。使用 Leaflet.js 实现交互式地图，支持城市搜索定位、核爆炸效应计算和可视化展示。

> ⚠️ **免责声明**：本模拟器仅供教育和研究目的使用。所有计算基于公开的物理模型和历史数据，结果仅供参考，不构成任何实际决策依据。

## ✨ 功能特性

### 🗺️ 交互式地图
- **多地图源支持**：高德地图、GeoQ、卫星图等多种地图源
- **城市搜索定位**：支持城市名称搜索，自动跳转到目标位置
- **层级城市列表**：按大洲/国家分组的城市选择器
- **坐标选择**：点击地图获取任意位置坐标
- **军事基地显示**：全球主要军事基地可视化

### 💥 核武器模拟
- **预设武器库**：45+ 真实核武器型号
  - 📜 **历史核武器**：小男孩 (15kt)、胖子 (21kt)、沙皇炸弹 (50Mt)、RDS-1、常春藤-麦克等
  - 🇺🇸 **美国**：W76、W78、W87、W88、B61、B83、民兵III、三叉戟II等
  - 🇷🇺 **俄罗斯/苏联**：SS-18 撒旦、白杨-M、布拉瓦、亚尔斯、萨尔马特等
  - 🇨🇳 **中国**：东风-41、东风-31AG、东风-5B、巨浪-2等
  - 🇬🇧 **英国**：蓝多瑙河、WE.177、三叉戟(英)
  - 🇫🇷 **法国**：AN-52、TN-75、老人星
  - 🇮🇳 **印度**：烈火-5、烈火-3
  - 🇵🇰 **巴基斯坦**：高里、沙欣-2
  - 🇰🇵 **朝鲜**：火星-15、火星-17
  - 🔬 **战术核武器**：大卫·克罗克特 (0.02kt)、SADM背包核弹、W54微型弹头
- **自定义当量**：支持 0.001kt - 100,000kt 自定义
- **爆炸高度**：地面爆炸、空中爆炸、高空爆炸
- **武器信息展示**：显示武器年份、国家、详细描述

### 📊 效应计算
- **火球半径**：物质完全气化区域
- **冲击波**：重度/中度/轻度破坏区 (20/5/2 psi)
- **热辐射**：三度烧伤范围
- **电离辐射**：500 rem 辐射区
- **电磁脉冲 (EMP)**：电子设备影响范围

### 📈 数据可视化
- **ECharts 图表**：专业的数据可视化
- **多标签面板**：基础、经济、基础设施、环境、健康
- **伤亡分布图**：饼图展示各区域伤亡
- **影响半径图**：柱状图对比各效应范围
- **经济影响分析**：GDP损失、恢复时间
- **基础设施评估**：医院、学校、交通、电力
- **环境影响**：辐射沉降、土地污染、碳排放
- **健康影响**：急性辐射、烧伤、创伤、长期癌症预测

### 👥 伤亡估算
- **人口密度**：可调整目标区域人口密度
- **时间段影响**：夜间/白天/高峰时段
- **国家特色数据**：80+ 国家的医疗、避难、建筑数据
- **2026年数据**：最新的人口和经济数据

### 🎨 界面设计
- **简洁高效**：移除所有动画，提升性能
- **深色主题**：护眼的深色界面
- **响应式设计**：适配桌面和移动设备
- **浮动工具栏**：便捷的操作按钮

## 🚀 快速开始

### 环境要求

- 现代浏览器 (Chrome 80+, Firefox 75+, Safari 13+, Edge 80+)
- 本地 Web 服务器 (可选，用于本地开发)

### 安装运行

```bash
# 克隆仓库
git clone https://github.com/badhope/bumb.git
cd bumb

# 方法1: Python 简易服务器
python -m http.server 8080

# 方法2: Node.js 服务器
npx serve

# 方法3: PHP 内置服务器
php -S localhost:8080

# 打开浏览器访问
# http://localhost:8080
```

### 使用说明

1. **选择武器**：点击 ⚙️ 按钮，选择预设武器或自定义当量
2. **选择目标**：
   - 点击 🎯 按钮，搜索城市名称
   - 或从下拉列表选择城市
   - 或直接点击地图选择位置
3. **开始模拟**：点击 💥 按钮引爆，查看爆炸效果和伤亡数据

## 🧮 计算模型

### 核心公式

#### 火球半径
```
R_fireball = 0.145 × Y^0.4  (km)
```

#### 冲击波半径
```
R_blast = k × Y^(1/3)  (km)

k 值:
- 20 psi (重度): k = 0.28
- 5 psi (中度):  k = 0.60
- 2 psi (轻度):  k = 1.00
```

#### 热辐射半径
```
R_thermal = 1.9 × Y^0.41  (km)
```

#### 电离辐射半径
```
R_radiation = 1.2 × Y^0.19  (km)
```
