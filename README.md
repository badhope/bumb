## рҹҺҜ What is BUMB?

**BUMB** (Nuclear Effects Simulator) is a professional-grade educational tool that simulates various nuclear explosion effects. 

Built with precision and clarity in mind, it provides:
- рҹ“Ҡ **Scientific calculations** based on public physical models
- рҹ—әпёҸ **Interactive maps** with global city data
- рҹ’Ҙ **Real weapons database** (45+ nuclear weapons)
- рҹ“Ҳ **Visual analytics** with ECharts

> *"Understanding the devastating effects of nuclear weapons is the first step toward a nuclear-free world."*

## рҹ§® Calculation Models

### Shockwave Effects

| Radius Type | Formula |
|-------------|---------|
| Fireball Radius | R = 0.145 Г— Y^0.4 km |
| Heavy Damage (20 psi) | R = 0.28 Г— Y^(1/3) km |
| Moderate Damage (5 psi) | R = 0.6 Г— Y^(1/3) km |
| Light Damage (2 psi) | R = 1.0 Г— Y^(1/3) km |

### Thermal Radiation

| Effect | Formula |
|--------|---------|
| Third-degree Burn Radius | R = 1.9 Г— Y^0.41 km |

### Ionizing Radiation

| Effect | Formula |
|--------|---------|
| 500 rem Zone | R = 1.2 Г— Y^0.19 km |

### EMP Effects

| Type | Formula |
|------|---------|
| Surface Burst | R = 10 + Y Г— 0.1 km |
| Airburst | R = 30 + Y Г— 0.2 km |
| High-altitude | R = 100 + Y Г— 0.5 km |

> Y = Yield in kilotons

## рҹӣ пёҸ Development

### Requirements

- Node.js >= 18.0.0
- pnpm >= 8.0.0

### Local Development

```bash
# Clone repository
git clone https://github.com/badhope/bumb.git
cd bumb

# Web version - Open index.html or use local server
python -m http.server 8080
# Visit http://localhost:8080

# Desktop version
cd nuclearsim-pro
pnpm install
pnpm start
```

### Build Desktop Version

```bash
cd nuclearsim-pro
pnpm build:win     # Windows
pnpm build:mac     # macOS
pnpm build:linux   # Linux
```
